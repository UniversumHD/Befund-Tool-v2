# Befund-Tool (Version 2.0.0)

**Eine lokale Anwendung zur schnellen Erstellung von Patientenbefunden für medizinische Praxen.**

---

## 📌 Beschreibung

Das **Befund-Tool** ist eine **lokale (vorläufig) macOS-Anwendung**, die speziell für Arztpraxen entwickelt wurde, um **Laborbefunde und andere medizinische Berichte** effizient zu erstellen. Mit vordefinierten **Textbausteinen** und einer **LaTeX-Vorlage** können Befunde in Sekunden generiert und als PDF exportiert werden – **ohne Internetverbindung**, für maximalen Datenschutz.

---

## ✨ Features

| Feature | Beschreibung |
|---------|--------------|
| **PDF-Generierung** | Textbausteine werden automatisch in eine LaTeX-Vorlage eingefügt und als PDF exportiert. |
| **Patientendaten** | Felder für **Name** und **Geburtsdatum** zur Personalisierung der Befunde. |
| **Textbaustein-Editor** | Bearbeitung und Verwaltung von Textbausteinen direkt im Programm. |
| **Verlaufsfunktion** | Übersicht über bereits erstellte Befunde für schnellen Zugriff. |
| **100% lokal** | Keine Cloud-Anbindung – alle Daten bleiben auf deinem Gerät. |

---

## 💻 Systemvoraussetzungen

- **Betriebssystem:** macOS (verifiziert für Version 15.5), in Zukunft auch Windows
- **Installation:** Auf Anfrage, noch keine Installationsanwendung verfügbar. Die Binary allein reicht nicht, da die Datenbank kofiguriert und Abhängigkeiten istalliert werden müssen

---

## 🛠 Verwendung

1. **Patientendaten eingeben:** Name und Geburtsdatum des Patienten eintragen.
2. **Textbausteine auswählen/bearbeiten:** Wähle aus vordefinierten Bausteinen oder passe sie an.
3. **PDF exportieren:** Mit einem Klick wird der Befund als PDF generiert und kann ausgedruckt oder digital weitergegeben werden.
4. **Verlauf nutzen:** Frühere Befunde sind im Verlauf einsehbar.
